# Bootstrap 3 for Meteor

[Bootstrap](http://getbootstrap.com) v3.2.0.

It works for the latest Meteor new [packaging system](http://docs.meteor.com/#usingpackages) introduced at 0.9.

## Installation

``` sh
$ meteor add pinglamb:bootstrap3
```
That's it. Enjoy.
